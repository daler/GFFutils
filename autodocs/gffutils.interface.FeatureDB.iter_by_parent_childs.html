gffutils.interface.FeatureDB.iter_by_parent_childs

FeatureDB.iter_by_parent_childs(featuretype='gene', level=None, order_by=None, reverse=False, completely_within=False)[source]

For each parent of type featuretype, yield a list L of that parent and all of its children ([parent] + list(children)). The parent will always be L[0].

This is useful for “sanitizing” a GFF file for downstream tools.

Additional kwargs are passed to FeatureDB.children(), and will therefore only affect items L[1:] in each yielded list.
